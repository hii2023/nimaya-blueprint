# Nimaya Blueprint

A single-page strategic pitch document built as an interactive web report. Created by Altaf Simavatwala as part of the application for the **Project Lead** role at Nimaya's GenAI Ready Naari Program.

## What It Does

Presents a detailed 90-day plan to expand the Nimaya Naari Program to Ahmedabad — from city intel and partner mapping to phase-wise budgets and risk mitigation.

## Key Features

- **10-section assignment report** covering every angle of the expansion plan
- **90-day launch blueprint** with clear phase visualization
- **3-phase milestone tracker** (Weeks 1-4, 5-8, 9-12)
- **15-day action plan** for the first sprint
- **Per-batch budget breakdown** for a cohort of 40 Naaris
- **Partner & role mapping** for Ahmedabad's ecosystem
- **City intelligence section** covering demographics, market fit, and outreach channels
- **Risk & struggle section** with mitigation strategies

## Tech Stack

- HTML, CSS, Vanilla JavaScript
- Single-file, no build step required

## Audience

Nimaya leadership and hiring panel reviewing the Ahmedabad expansion pitch.
